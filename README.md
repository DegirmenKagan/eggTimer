# eggTimer
eggTimer app is for cooking your eggs however you like perfectly.
It has 3 different options which are 
<ul>
<li>Soft: 3 minutes</li>
<li>Medium: 7 minutes</li>
<li>Hard: 12 minutes</li>
</ul>
